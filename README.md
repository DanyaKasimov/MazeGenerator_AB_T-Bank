# Генератор лабиринтов

## Алгоритмы генерации лабиринта

Приложение поддерживает два алгоритма генерации лабиринта:

* Алгоритм Прима
* Алгоритм Краскала

## Алгоритмы поиска пути

Приложение поддерживает два алгоритма поиска пути:

* Алгоритм BFS
* Алгоритм Дейкстры
